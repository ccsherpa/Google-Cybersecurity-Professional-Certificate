📂 Featured Projects
1. NIST CSF Incident Analysis: DoS Attack
Path: /NIST-Incident-Analysis/
• Scenario: A multimedia company experienced a 2-hour network compromise due to a Denial of Service (DoS) attack.
• Vulnerability: A malicious actor exploited an unconfigured firewall to flood the network with ICMP packets.  
• Core Actions:
• Identify: Documented the loss of network resources and identified the ICMP flood as the root cause.  
• Protect: Deployed an IDS/IPS system to filter traffic based on suspicious characteristics.  
• Detect: Implemented Source IP address verification on the firewall to identify spoofed packets.  
• Respond/Recover: Focused on isolating affected systems and prioritizing the restoration of critical network services.
